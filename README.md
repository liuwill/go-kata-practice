### Script

```shell
# 运行单元测试
make test

# 单元测试并且生成测试覆盖报告
make coverage

# 单元测试并且测试覆盖HTML页面
make coverhtml
```

#### 运行单个测试

run.test.sh脚本，参数一设置目录

```shell
./run.test.sh rain
```

#### docker 环境运行

```shell
# 创建docker容器
./docker.sh install

# 进入docker容器
./docker.sh enter

# 在容器中运行命令
cd /go/src/go-kata-practice
make coverhtml

# 或者
./docker.sh test
```
